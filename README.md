# COE Analytics Dashboard

## Overview

The **COE Analytics Dashboard** is an interactive **Streamlit-based analytics application** designed to monitor and analyze **Center of Excellence (COE) initiatives**.

The dashboard provides insights into **initiative performance, KPI achievement, business value generated, and project timelines**, enabling stakeholders to quickly understand the impact and health of ongoing initiatives.

This project simulates a real-world **COE program monitoring system** commonly used in enterprise analytics and consulting environments.

---
## Dynamic Dataset Generation

The dashboard includes a **"Generate New Dataset"** button in the sidebar that creates a new synthetic dataset of **100 COE initiatives** each time it is clicked.

This allows the dashboard to **simulate dynamic data updates**, automatically refreshing all metrics, charts, and timelines to demonstrate how the analytics system performs with varying data scenarios.

## Key Features

### KPI Monitoring

Track high-level program metrics including:

* **Total Initiatives**
* **Average KPI Achievement**
* **Percentage of On-Track Initiatives**
* **Total Business Value Generated (₹)**

### Initiative Status Analytics

Visualize initiative progress through:

* **Status Distribution Charts**
* **Initiative Performance Breakdown**

### Business Value Insights

Analyze the business impact generated by initiatives:

* **Distribution of Business Benefits**
* **Total Benefit by Initiative Owner**
* **Top Performing Initiatives**

### Initiative Timeline (Gantt Chart)

A visual timeline of all initiatives showing:

* Start and end dates
* Initiative duration
* Status-based coloring

### Initiative Health Score

Each initiative is assigned a **Health Status** based on KPI performance and project status:

* 🟢 **Green** – Healthy initiative (high KPI performance)
* 🟡 **Yellow** – Moderate performance
* 🔴 **Red** – Delayed or underperforming initiative

### Interactive Filtering

Users can dynamically filter the dashboard by:

* **Initiative Status**
* **Initiative Owner**

### Dynamic Data Simulation

The dashboard includes a **synthetic dataset generator** which allows users to create **new datasets of 100 initiatives** with a single click.

This enables the dashboard to simulate **dynamic data refresh scenarios**, making the analytics system more robust.

## Installation

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/biocon_analytics_dashboard.git
```

Navigate into the project:

```
cd biocon_analytics_dashboard
```

Create a virtual environment:

```
python -m venv analytics_dashboard
```

Activate the environment:

**Windows**

```
analytics_dashboard\Scripts\activate
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## Running the Dashboard

Start the Streamlit application:

```
streamlit run dashboard/app.py
```

The dashboard will open in your browser.
